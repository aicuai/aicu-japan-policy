# AICU Japan 契約・モラルガイドライン規約集

このリポジトリは、AICU Japan 株式会社が公開・整備する、契約書、業務条件、モラルガイドライン、AI利用規約、制作・開発に関するポリシー集です。

AICU AIDX Lab では、お客様、開発者、クリエイティブAI分野のクリエイターの皆様にとって、AI/DX時代の業務や制作が「三方よし」となることを目指し、契約・権利・責任範囲・AI利用・制作倫理に関する考え方を公開で整備しています。

本リポジトリの文書は、AI/DX支援、生成AI活用、準委任業務、AIクリエイティブ制作、開発支援、PoC、教育、研究、コンテンツ制作等において、関係者が安心して協議・契約・制作を進めるための参考資料として提供するものです。

## aicu.jp/policy への公開

本リポジトリの一部の文書は https://aicu.jp/policy に自動公開されます。

- 公開対象は [publish.json](publish.json) で管理します（ファイル・URLスラッグ・タイトル）。
- main への push 時に GitHub Actions（[notify-aicujp.yml](.github/workflows/notify-aicujp.yml)）が [app-aicujp](https://github.com/aicuai/app-aicujp) のデプロイを起動し、ビルド時に本リポジトリから文書を取得して静的ページを生成します。

## コメント・PRについて

本リポジトリへのコメント、Issue、Pull Request を歓迎します。

特に、以下のような観点からのご意見を歓迎します。

- 法律専門家、弁護士、企業法務担当者によるレビュー
- AI/DX案件、準委任契約、開発支援、制作委託に関する実務上の改善提案
- 生成AI、OSS、オープンモデル、AIクリエイティブ制作に関する権利処理・モラル・運用上の提案
- 発注者、受託者、エンドユーザー、開発者、クリエイターのいずれかに過度な負担が偏らないための改善提案
- わかりにくい表現、過度に硬い表現、実務で使いづらい表現の修正提案

法律専門家によるレビュー、コメント、改善提案、PRは歓迎します。  
ただし、PRの改修は義務ではなく、判断は AICU Japan 株式会社が行います。個別案件に関する法律相談や紛争相談は、本リポジトリ上では対応いたしません。

## 基本方針「三方よし」

AICU Japan 株式会社は、AI/DX時代の契約・制作・開発において、発注者、受託者、エンドユーザー、開発者、クリエイターのそれぞれが合理的な利益を得られることを重視します。

本リポジトリの文書は、一方的な権利確保や責任転嫁を目的とするものではありません。  
プロジェクト内での利用を妨げず、同時に、既存IP、非公開ノウハウ、データ、スクリプト、テンプレート、創作物、AI利用上の責任範囲が不明確なまま拡散しないようにすることを目的としています。

AIを活用した業務や制作が、関係者にとって持続可能で、透明で、実装しやすいものとなるよう、公開で整備していきます。
